# 🚀 Lista 13 — Estrutura de Dados (Matrizes e Registros)

Dados do mundo real muitas vezes são tabelas ou fichas. Vamos praticar matrizes (linhas e colunas), registros (campos variados) e a combinação dos dois.

---

## 🛠️ O Algoritmo da Entrega

1. Resolva os exercícios no **Portugol Studio**.
2. Crie o repositório público `una-algprog-lista13`.
3. No `README.md`, explique a diferença entre uma **matriz** e um **vetor de registros**.

---

## 📝 Lista de Exercícios

??? abstract "Exercício 1: Soma de Matriz"
    Leia uma matriz 3x3 de inteiros e exiba a soma de todos os elementos.

??? abstract "Exercício 2: Diagonal Principal"
    Leia uma matriz 3x3 e exiba apenas os elementos da diagonal principal (onde linha = coluna).

??? abstract "Exercício 3: Ficha de Aluno (Registro)"
    Crie um registro `aluno` (nome, matrícula, média). Leia os dados e exiba um boletim formatado.

??? abstract "Exercício 4: Agenda de Contatos"
    Use um vetor de 5 registros (nome e telefone). Preencha e liste todos os contatos.

??? abstract "Exercício 5: Maior da Matriz"
    Leia uma matriz 2x2 e informe o maior valor e em qual posição (linha, coluna) ele está.
