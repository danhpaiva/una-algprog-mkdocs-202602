# 🚀 Lista 10 — Repetição: FOR e FOREACH

Quando você sabe **quantas vezes** repetir, o laço `Para` é mais elegante e seguro. Vamos praticar contadores, somatórios e laços aninhados.

---

## 🛠️ O Algoritmo da Entrega

1. Resolva os exercícios no **Portugol Studio**.
2. Crie o repositório público `una-algprog-lista10`.
3. No `README.md`, explique quando usar `Para` (for) e quando usar `Enquanto` (while).

---

## 📝 Lista de Exercícios

??? abstract "Exercício 1: Tabuada com For"
    Reescreva a tabuada usando `Para` em vez de `Enquanto`.

??? abstract "Exercício 2: Soma dos Pares"
    Some todos os números pares de 1 a 100 e exiba o resultado (use `passo`).

??? abstract "Exercício 3: Fatorial"
    Leia um número `n` e calcule seu fatorial (`n! = 1 * 2 * ... * n`).

??? abstract "Exercício 4: Contagem Regressiva"
    Faça uma contagem regressiva de 10 até 0 e, no final, exiba "Lançar!" (use `passo -1`).

??? abstract "Exercício 5: Tabuadas (Laço Aninhado)"
    Use laços aninhados para exibir as tabuadas de 1 a 5, todas de uma vez.
