# 🚀 Lista 09 — Repetição: WHILE e DO-WHILE

Chegou a hora de fazer o computador repetir tarefas sem cansar. Cuidado com o **loop infinito**: todo laço precisa caminhar em direção ao seu fim!

---

## 🛠️ O Algoritmo da Entrega

1. Resolva os exercícios no **Portugol Studio**.
2. Crie o repositório público `una-algprog-lista09`.
3. No `README.md`, explique a diferença entre `Enquanto` e `Repita` (qual testa antes e qual testa depois).

---

## 📝 Lista de Exercícios

??? abstract "Exercício 1: Tabuada"
    Leia um número e exiba sua tabuada de 1 a 10 usando `Enquanto`.

??? abstract "Exercício 2: Soma até Parar (Sentinela)"
    Leia números do usuário e vá somando. Pare quando ele digitar `0`. Exiba a soma total.

??? abstract "Exercício 3: Menu com Repita"
    Crie um menu que fica repetindo até o usuário escolher "Sair". Use `Repita ... Ate`.

??? abstract "Exercício 4: Validação de Nota"
    Peça uma nota de 0 a 10. Enquanto o valor for inválido, peça novamente.

??? abstract "Exercício 5: Contador de Pares"
    Conte quantos números pares existem de 1 a 50 usando um laço.
